## Project Reality

- The app is real and runnable today.
- The parser is heuristic and layout-sensitive.
- The canonical model is the main internal contract.
- The `.esx` package is deterministic and validated, but not a native proprietary `XACTDOC.ZIPXML` writer.
- The project is local-first and does not rely on cloud parsing services.
